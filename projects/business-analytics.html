India Crime Rate Dashboard

Tableau Predictive Analytics

Project Overview

Developed an interactive dashboard to gain a deeper understanding of the criminal activities in India. This project utilized data visualization techniques to identify patterns and trends in crime rates across different regions and time periods.

Key Achievements

  • Created an interactive Tableau dashboard that visualizes crime data across Indian cities
  • Implemented filtering capabilities allowing users to explore data by crime type, region, and time period
  • Identified significant correlations between age group and crime types

Technologies Used

  • Tableau for data visualization and dashboard creation
  • Predictive modeling techniques
  • Data cleaning and preparation tools
